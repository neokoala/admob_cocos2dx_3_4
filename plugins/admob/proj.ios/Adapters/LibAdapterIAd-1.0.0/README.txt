====================================================
Ad Network Adapter for Google Mobile Ads SDK for iOS
====================================================

This is an adapter to be used in conjunction with the Google Mobile Ads SDK
for iOS.

Requirements:
- Xcode 5.1 or later
- Runtime of iOS 6.0 or later
- Associated Ad Network SDK

Instructions:
- Add the included binary (.a library) into your XCode project with 
its associated Ad Network SDK.
- Add any additional frameworks required by the associated Ad Network SDK.
- Enable the Ad network in the Ad Network Mediation UI.

The latest documentation and code samples for the Google Mobile Ads SDK are 
available at:
https://developers.google.com/mobile-ads-sdk/
